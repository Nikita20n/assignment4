1.what is oop?list oop concepts
Object-oriented programming (OOP) is a computer programming model that organizes software design around data, or objects, rather than functions and logic. An object can be defined as a data field that has unique attributes and behavior.
list oop concepts:
Class
Objects
Encapsulation
Abstraction
Polymorphism
Inheritance

oop:
1.it is a bottom- up approch.
2.it concentrated on data rather than functions.
3.data is used by a associated function within the class.
4.it is more secured language because external function can't access another data.
5.example:c++,java,visual basic,python,php.

1.it is a top down approch.
